# str-similar
calculate similar of chinese strings

### 支持中文字符串相似度检测
```text
yhn@yhnMBP similar-cn % ./bin/str-similar.ts compare 太和 大和
similarity 0.92885
yhn@yhnMBP similar-cn % ./bin/str-similar.ts compare 啥 哈    
similarity 0.92448
yhn@yhnMBP similar-cn % ./bin/str-similar.ts compare 你 我
similarity 0.52308
```
